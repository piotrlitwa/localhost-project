# Assets - Materialy do postow

## Struktura folderow

```
assets/
├── produkty/       # Opisy produktow (.txt, .md, .pdf)
├── zdjecia/        # Zdjecia do postow (.png, .jpg, .webp)
└── inspiracje/     # Referencje, screenshoty, linki (.md, .txt, obrazki)
```

## Jak uzywac

### produkty/
Wrzucaj tu pliki z opisami produktow/uslug. Kazdy plik = jeden produkt.
- `produkt-nazwa.txt` - opis, cechy, cena, link
- `produkt-nazwa.md` - bardziej rozbudowany opis z formatowaniem

### zdjecia/
Zdjecia produktow, grafiki, loga - wszystko co moze pojsc do posta.
- Obslugiwane formaty: PNG, JPG, JPEG, WebP, GIF, BMP
- Nazywaj pliki opisowo: `buty-sportowe-01.jpg`, `logo-firmy.png`

### inspiracje/
Screenshoty postow konkurencji, linki do artykulow, notatki.
- Wszystko co moze pomoc w tworzeniu lepszych tresci.
